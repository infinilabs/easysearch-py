[[installation]]
== Installation

The Python client for {es} can be installed with pip:

[source,sh]
-------------------------------------
$ python -m pip install easysearch
-------------------------------------

If your application uses async/await in Python you can install with the `async` 
extra:

[source,sh]
--------------------------------------------
$ python -m pip install easysearch[async]
--------------------------------------------

Read more about 
https://github.com/infinilabs/easysearch-py/en/master/async.html[how to use Asyncio with this project].
